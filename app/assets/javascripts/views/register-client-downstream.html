• Enable satellite-tools repository if you want to use client tools such as katello-agent or puppet:

    For Red Hat Enterprise Linux 8 and newer:

    subscription-manager repos --enable=satellite-tools-$SATELLITE_VERSION-for-rhel-$RHEL_MAJOR_VERSION-x86_64-rpms

    For example:

    subscription-manager repos --enable=satellite-tools-6.7-for-rhel-8-x86_64-rpms

    For Red Hat Enterprise Linux 7 and older:

    subscription-manager repos --enable=rhel-$RHEL_MAJOR_VERSION-server-satellite-tools-$SATELLITE_VERSION-rpms

    For example:

    subscription-manager repos --enable=rhel-7-server-satellite-tools-6.7-rpms
